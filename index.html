Lessons

Lesson 01 - Python basics

The basics of the Python programming language.

Topics covered: data types, lists, sets, dictionaries, functions. NumPy, Matplotlib.

Open GitHub folder

Supplementary materials:

  • Official Python tutorial
Lesson 02 - The basics of RL

The basics of reinforcement learning and the most important definitions (state, action, reward). The behaviour of random agents.

Topics covered: the RL framework, MDP, reward hypothesis.

Open GitHub folder
Lesson 03 - Q-learning

The various groups of reinforcement learning algorithms. The motivation behind value-based methods & the basics of Q-learning.

Topics covered: types of RL algorithms, discount factors, exploration vs exploitation, value-based methods.

Open GitHub folder
Lesson 04 - OpenAI Gym

We explore the capabilities of the OpenAI Gym library that has lots of different environments that we can use to train RL agents.

Topics covered: OpenAI Gym, types of environments.

Open GitHub folder
Lesson 05 - Unity

We familiarize ourselves with the Unity game engine, learning the basics of 2D and 3D objects and environments.

Topics covered: basics of Unity, 2d and 3D modeling, basic concepts.

Open GitHub folder

Resources:

  • Unity download page
  • Unity Asset Store
  • Submarine prefab
  • Treasure prefab
  • Water texture prefab
  • Standard Assets (for the water prefab)
Lesson 06 - Unity ML-Agents Toolkit

We dive deeper into reinforcement learning by exploring the capabilities of the Unity ML-Agents Toolkit plugin.

Topics covered: basics of Unity ML-Agents Toolkit, RL in Unity, key definitions & workflow.

Open GitHub folder

Resources:

  • Check out the installation guide on the e-learning website

  • Running ML Agents from the command line: mlagents-learn --torch-device "cpu"

Lesson 07 - More complex environments

We dive deeper into using the ML-Agents Toolkit for tackling reinforcement learning problems in Unity. We modify the previous environment by using continuous actions which will make training the agent harder.

Topics covered: dealing with continuous actions & YAML config files.

Open GitHub folder

Resources:

  • ML-Agents PPO configuration notes

  • Running ML Agents from the command line with the YAML file: mlagents-learn .\TreasureSearch.yaml --run-id="Run 1"

Lesson 08 - Tips for training your agents

We discuss what best practices should be followed when training RL agents in Unity.

Topics covered: training tips & limitations.
